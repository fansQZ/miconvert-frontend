## 搭建过程

~~~shell

第二步：初始化vite，来创建工程，按照vue官网步骤，但是全部选no
https://cn.vuejs.org/guide/quick-start.html
npm install  //安装需要的包
npm run dev  //运行开发环境
npm run build //运行生产环境
npx vite build //打包
第三步：安装router
https://router.vuejs.org/zh/introduction.html
npm install vue-router@4
//创建router在vue3中使用
